This tool is a first working draft. It's still missing a lot of features...

You are welcome to contribute for making it better : github.com/fddsfds

Dataset

EU12 is a 12km resolution regional weather model over Europe. It's run 4 times a day on the Open Meteo Foundation's supercomputer.

This model produces hourly data for the next 72 hours.

The data is available under Open Data Commons Attribution Licence (ODC-By). Which means the data is freely available for any use, as long as you attribute the source.
